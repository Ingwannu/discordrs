## 요약

- Builders: Components V2/Modal 구조를 플루언트하게 생성
- Gateway: heartbeat/resume/reconnect 포함 런타임
- Parsers: 인터랙션/모달을 타입 안전하게 파싱
- HTTP + Helpers: 응답 API를 간결하게 제공
- Interactions Endpoint: Ed25519 검증 기반 HTTP 콜백 처리
